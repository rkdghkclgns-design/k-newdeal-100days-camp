# K-뉴딜 100일 캠프 — 랜딩페이지

디벨로켓 × 넥슨이 함께하는 **「전직 전 초보자를 위한 100일 캠프」** 모집 랜딩페이지입니다.
K-뉴딜 청년 디지털 일자리 지원사업의 일환으로, MMORPG 퀘스트 메타포(초보자 마을 · 전직 · 스탯 · 골드)를
활용해 부담 없이 공감·전환을 유도하는 구조로 설계되었습니다.

## 실행 방법

정적 페이지이므로 별도 빌드가 필요 없습니다.

- 가장 간단하게: `K-뉴딜 100일 캠프.html` 파일을 브라우저로 엽니다.
- 권장(영상·폰트·이미지 슬롯이 안정적으로 로드됨): 폴더에서 로컬 서버 실행 후 접속
  ```bash
  # Python 3
  python -m http.server 8000
  # 또는 Node
  npx serve .
  ```
  브라우저에서 `http://localhost:8000/K-뉴딜 100일 캠프.html` 접속

## 파일 구조

```
렌딩페이지/
├─ K-뉴딜 100일 캠프.html   # 페이지 셸 (섹션 컨테이너 + 스크립트 로드)
├─ assets/
│  ├─ styles.css            # 디자인 시스템 (컬러·타이포·레이아웃·반응형)
│  ├─ content.js            # 모든 카피·항목의 단일 데이터 출처(DEFAULT_CONTENT)
│  ├─ render.js             # 데이터 → DOM 렌더 엔진 (인라인 수정 바인딩 포함)
│  ├─ admin.js              # 관리자 모드 (인라인 수정·항목 CRUD·localStorage 저장)
│  ├─ app.js                # 인터랙션·초기화 (스크롤·FAQ·공감 애니메이션·고정바·영상)
│  ├─ image-slot.js         # 드래그&드롭 이미지 슬롯 커스텀 엘리먼트 (넥슨 카드)
│  ├─ hero.mp4              # 히어로 카드 영상
│  └─ logo-*.png            # 디벨로켓 · 넥슨 · 메이플스토리 로고
└─ README.md
```

### 콘텐츠는 코드와 분리되어 있습니다
페이지의 모든 문구·항목은 `assets/content.js`의 `DEFAULT_CONTENT` 객체에서 렌더됩니다.
문구를 코드에서 직접 바꾸려면 이 파일만 수정하면 됩니다. (관리자 모드로 수정하면 브라우저 localStorage에 병합 저장됩니다.)

## 페이지 구성 (14개 섹션)

히어로(다크 RPG 퀘스트 카드 + 영상) → 프로그램 한 줄 정의 → 공감 유도(초보자 마을 퀘스트 로그) →
솔루션(지역 한정 안내) → 커리큘럼(6개 STEP, STEP 6은 ★히든 퀘스트) → 수업 혜택(고급 노트북 대여 ·
Claude 프리미엄 시트 · 월 50만원 학습 장려금) → 넥슨 특별 혜택(콘테스트 · 멘토링 · 사옥 견학 +
메이플스토리 공식 리소스 배너) → 지원 자격(지역 · 구직/미취업 · 선착순) → 중간 CTA → FAQ → 신청 폼 → 푸터

전환 구조: **상단 고정 CTA + 하단 고정 신청바 + 중간 CTA**가 모두 페이지 내 신청 폼으로 스크롤됩니다.

## 관리자 모드

상단 네비게이션 우측의 **🔒 자물쇠 아이콘**을 클릭 → 비밀번호 입력으로 진입합니다.

- 비밀번호: `0000` (변경: `assets/admin.js`의 `ADMIN_PASS`)
- 기능:
  - 모든 문구(헤드라인 · 본문 · CTA) **클릭 인라인 수정**
  - 상단 메뉴 · STEP · 혜택 카드 · FAQ · 지역 항목 **추가(＋) · 삭제(✕) · 순서변경(▲▼)**
  - 수정 내용 **브라우저 자동 저장**(localStorage), **초기화** · **JSON 내보내기** 지원
- 종료: 상단 관리자 바의 **편집 종료** 또는 자물쇠 아이콘 다시 클릭

## 참고

- **넥슨 혜택 카드 3종 이미지**는 드래그&드롭 이미지 슬롯으로 비워져 있습니다.
  관리자가 이미지 파일을 끌어다 놓으면 채워집니다(영속 저장은 호스트 런타임 환경에서 동작).
- **신청 폼**은 현재 프론트엔드 데모입니다(제출 시 완료 화면 표시). 실제 접수 연동이 필요하면
  `assets/app.js`의 `submit` 핸들러에 전송 로직을 추가하세요.
- 폰트는 CDN(Pretendard · Galmuri 픽셀폰트)에서 로드되므로 인터넷 연결이 필요합니다.
